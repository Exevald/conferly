# conferly
Service for managing and viewing conferences and meetups

# Сейчас реализовано:

## Страницы

- **`/`** - Главная страница
  - Отображение списка мероприятий с возможностью переключения режимов просмотра (rows, grid, list)
  - Фильтрация и поиск мероприятий
  - Счетчик мероприятий

- **`/auth`** - Страница аутентификации
  - Режим входа (`?mode=login`) - форма логина
  - Режим регистрации (`?mode=register`) - форма регистрации

- **`/components`** - Страница демонстрации компонентов дизайн-системы
  - Интерактивная демонстрация всех компонентов ДС
  - Боковая панель для навигации между компонентами

- **`/conferences`** - Страница списка конференций
  - Временно перенаправляет на главную страницу

- **`/conferences/[id]`** - Страница детальной информации о конференции
  - Заглушка (в разработке)

- **`/projects`** - Страница управления проектами
  - Панель навигации с категориями и списком проектов
  - Рабочее пространство управления
