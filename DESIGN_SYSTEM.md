# Дизайн-система Conferly

## Основные принципы

Дизайн-система построена на основе CSS-переменных и модульных компонентов. Используется цветовая палитра с акцентным цветом `#FA5D20` (primary).

## Цветовая палитра

- **Primary**: `#FA5D20` - основной акцентный цвет
- **Text**: 
  - Primary: `#323537`
  - Secondary: `#6C757D`
  - Muted: `#ADB5BD`
- **Background**:
  - Primary: `#FFFFFF`
  - Secondary: `#F8F9FA`
  - Accent: `#FA5D20`
- **Border**:
  - Light: `#E9ECEF`
  - Medium: `#DEE2E6`
  - Dark: `#ADB5BD`
